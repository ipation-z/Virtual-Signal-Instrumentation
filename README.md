Virtual Signal Instrumentation （虚拟信号源与示波器上下位机系统）本项目是一款基于 STM32 单片机硬件、Python 虚拟模拟器与 PySide6+PyqtGraph 上位机开发的综合性虚拟仪器系统。 系统集成了四通道时域示波器与多波形讯号发生器，支持全双工高速序列埠通讯，具备高即时性、强抗干扰性与流畅的GUI互动体验。📂 项目架构与模块说明本仓库包含以下三大核心程序部分：upper_computer.py （ 专业级上位机）基于 PySide6 （Qt for Python） 与 PyqtGraph 打造。采用多线程异步通信架构，将「GUI 界面渲染」与「底层序列埠数据吞吐」彻底隔离，保障 4 通道数据并行高频刷新时不卡顿、不闪退。整合专业示波器控制（AUTO/NORMAL/SINGLE 触发、时基/电位调节）与信号发生器配置面板。main.c / usbd_cdc_if.c （STM32 单片机硬件固件）基于STM32F103（或其他支持USB FS芯片）开发。输入端：利用TIM定时器触发4通道ADC扫描采样，并透过DMA搬运至内存缓冲区，经由USBCDC（VCP虚拟序列端口）高速上传。输出端：接收上位机控制影格，利用动态定时器配置计数周期，透过TIM+DMA输出高速 PWM 模拟 DAC 来实现高精度波形发生器。单片机虚拟仿真器 （Simulator）整合于上位机调试系统或独立仿真脚本中。支持在无 STM32 实体硬件的情况下，利用 Python 的虚拟序列端口对抗（如 对接），模拟 STM32 底层寄存器级的数据流水与状态机响应，供纯软件验证和通讯协议除错。com0com✨ 核心功能特性四通道时域示波器支持 4 个独立通道同时采样，具备硬件级波形同步。支持上升沿/下降沿触发，自定义触发电位（Trigger Level），拥有专业级波形锁定能力。精细优化底噪，解决了多通道 ADC 悬空时的串扰（Crosstalk）与天线效应。多波形讯号发生器支持输出四大标准波形：正弦波、方波、三角波、锯齿波。动态可调参数：频率（1Hz - 100kHz）、峰峰值（Vpp 0 ~ 3.3V）。高可靠全双工通讯链路下行（控制）：严格的 状态机解包架构，解决断包、粘包问题。0x5A 0xA5Windows 独占避坑机制：重构通讯对象生命周期，控制影格完全由背景 通讯线绪动态代理写入（），完美绕过Windows硬件底层独占引发的错误。SerialReadThreadwritePermissionError(13) 拒絕訪問🛠️ 开发环境与依赖配置1. 上制机和模拟器环境（Python）环境需求:P ython 3.8 或更高版本快速安装依赖 ：Bashpip install PySide6 pyqtgraph numpy pyserial
